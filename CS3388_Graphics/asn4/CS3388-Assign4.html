CS3388 ASSIGNMENT 4

CS3388 ASSIGNMENT 4

Due date: Friday December 6 2013
Weight: 10% of final mark

This assignment consists of the writing of a simple ray-tracing program that creates images of scenes containing a number of simple generic objects. The ray tracing program must be able to:

  1. deal with spheres and planes of finite extent as generic objects

  2. implement local shading

  3. implement shadowing through the use of feeler rays

  4. capture user input to place and transform spheres and planes in the scene (in other words, the user must be able to specify the type of objects (spheres, planes) and enter a transformation matrix for each object, along with surface properties (color, specularity, etc)

  5. capture user input to place the viewpoint within the scene and define a light source

  6. Render the image on the screen

There is a 5 term-mark bonus for a correct recursive implementation of light refraction or reflection, or both.
