This assignment consists of the design and implementation of a program that allows a user to define and render 3D objects defined around an axis of symmetry. Your program will read 3D points forming a 2D profile defined in the (X,Z) plane, rotate this profile around the Z-axis by a small angle, collect these newly formed points and create a polygonal mesh made up of triangles. Your program will perform the following:

  1. Read the 3d points forming the 2D profile from this file.

  2. Apply an appropriate number of rotations of the points around the Z-axis. While this is performed, compute and storage triangular faces in a counterclockwise fashion to form a polygonal mesh of the object. Use a data structure that is consonant with that of the Course Textbook on page 276.

  3. The next part of the program is to render the object. In order to accomplish this correctly, the program must:

    • Define at least one light source with ambient, diffuse, and specular components specified.

    • Define material properties for the object being created.

    • Implement hidden surface removal but not back face removal, because for some objects, back faces may be visible.

    • Animate the object by defining a camera and moving it around the object, in a fashion of your choice.

    • Animate the lighting by independently moving the light around the object, on a path different from the camera.

    • The object must look as realistic as possible.

  4. The program will terminate when the user presses 'Q'.
